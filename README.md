# CISC226-Project-Group8
Group project for CISC 226 - Game design. We are developing an aim trainer targeted towards kids called "Pizzapocalypse - Aim Trainer"

The goal of this game is to give younger children aged 7-12 an engaging way to improve their aiming skills, without the immense pressure of competitive online shooters. This will allow them to build easily transferrable skills that they can bring forth with them into their future endevors.

The first release for this game is now live, including a world with 7 levels, along with 3 selectable robots with varying skills.
